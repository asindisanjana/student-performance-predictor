🎓 Student CGPA Predictor

This project predicts a university student's final CGPA based on study habits, attendance, and lifestyle factors using Machine Learning.

🚀 Features
- Predict CGPA using Random Forest Regression
- Interactive web app built with Streamlit
- User-friendly interface with dropdowns and sliders

📊 Dataset
University Student Performance and Habits Dataset (Kaggle)
Student_data.csv

🧠 Model
- Random Forest Regressor
- Achieved high accuracy in predicting CGPA

🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
